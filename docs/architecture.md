# توثيق معمارية سلّمها

## نظرة عامة

سلّمها منصة SaaS لتسليم المنتجات الرقمية لمتاجر سلة. تتكون من:

1. **Laravel API** - REST API للمصادقة، إدارة المنتجات، المخزون، الطلبات، والتسليم
2. **Next.js Frontend** - لوحة تحكم للتاجر + صفحة تسليم للعميل

## الأدوار

- **Admin** - مسؤول المنصة
- **Merchant** - صاحب المتجر
- **Customer** - العميل النهائي (بدون حساب)

## تدفق التسليم

1. التاجر ينشئ طلب → يتولد Delivery Link
2. العميل يفتح `/d/{store_slug}/{token}`
3. العميل يدخل رقم الطلب + رقم الجوال
4. النظام يتحقق → يعرض المنتج الرقمي
5. يسجل في delivery_logs

## الأمان

- جميع الأكواد مخزنة مشفرة (AES-256)
- روابط التسليم محمية بـ UUID v4
- Rate Limiting على محاولات التحقق
- Pessimistic Locking لمنع تكرار استخدام الكود
